3. 依任務走，不要只查一本書

想把作文寫得高級：先查 Well-Spoken / Verbal Advantage / Advanced Vocabulary，再用 Collocations 檢查搭配，最後用 IELTS/AP writing routes 看句子是否能放進論證。
想要道地：先查 Collocations / Phrasal Verbs / Idioms，再查 Advanced Everyday English / spoken notes；如果是網路語氣，再看 Because Internet。
想寫 argument：先查 weak-argument / see-your-point / offer-thoughts route，再查 Writing Arguments / IELTS sample answers / AP essay。
想要文學感：先查 fragment deck / Wilde / Twain / Austen，再查 Oxford literary terms / Norton anthology / quotations by subject。
