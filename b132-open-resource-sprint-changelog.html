What changed structurally

  1. Open/free/public resources are now separated by readiness: chunked, downloaded raw, metadata-only, and route-only license-check.
  2. Official English exams are registered as route/calibration resources, not confused with public-domain full text.
  3. Project Gutenberg public-domain texts can now feed quote-led writing prompts, speaking topics, style imitation, and cross-source context packs.
  4. Every public page was checked for public HTTP access and mobile horizontal overflow.
