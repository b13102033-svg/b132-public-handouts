Turn a fragment into an answer

Use one source fragment as a thinking tool. Do not just quote it; make it do work inside your answer.

1. Claim

Take a position that can be argued, not a slogan.

Although ___, I think ___ because ___.

2. Evidence

Point to one fragment from the quick route or evidence library.

One fragment suggests that ___.

3. Reasoning

Explain why that fragment changes the way we should think.

This matters because it shows that ___ is not simply ___, but ___.

4. Turn back

Return to the question with a clearer criterion.

So the real question is not whether ___, but when ___ becomes ___.
Two-minute drill: choose one quick-route fragment, speak for 45 seconds using steps 1-3, then write one improved sentence using step 4.

Speaking task

Give a 90-second answer to the question. Start with a concession, use one source fragment, then connect it to a real situation.

  1. State a nuanced position.
  2. Use one fragment as evidence.
  3. Give a school, family, media, or personal example.
  4. End with a clear criterion: when your position works and when it fails.

Answer shape

Although... introduce the opposite side.

What matters is... name your criterion.

This is not because..., but because... sharpen the logic.
